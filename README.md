# 威少投资智脑系统

自动化投资追踪与分析系统

## 功能模块

### 1. 盘面追踪
- A股主要指数估值（沪深300、中证500、创业板）
- 市场情绪指标（涨跌停比、成交量）
- 风格轮动信号

### 2. 行业追踪
- 持仓相关行业ETF估值
- 行业资金流向
- 政策/消息面监控

### 3. 个股追踪
- 持仓股技术面预警
- 消息面监控
- 盈利/亏损提醒

### 4. 聪明钱追踪
- 北向资金（外资）动向
- 主力资金流向
- 融资融券变化
- ETF份额变动

### 5. 跨境投资
- 港股优质标的
- 美股ETF配置
- REITs/黄金

## 技术架构

```
investment系统/
├── data/           # 数据存储
├── scripts/        # 爬虫脚本
├── reports/        # 生成的报告
└── app.py          # 可视化界面
```

## 使用方式

1. 启动可视化界面: `python app.py`
2. 定时任务: 每天8:00自动分析并通过飞书推送
3. 手动查询: 访问界面查看实时数据
